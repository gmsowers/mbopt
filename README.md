# Mbopt library

This is a C++ library for formulating mass balance models of chemical processes and solving them with [Ipopt](https://github.com/coin-or/Ipopt). It is designed to be called from a Lua script. It also provides a binary that can be invoked with a Lua script as an argument.

This package is unregistered, which means it's not in the Julia package registry. If you happen to discover it and want to use it, do this:

[Mbopt documentation](https://gmsowers.github.io/mbopt/).
